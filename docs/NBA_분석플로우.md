### 1단계: 문제 정의

**목적**: 분석의 목적과 핵심 질문 명확화

**주요 활동:**
- 핵심 문제 정의: "NBA 경기에서 승리를 결정하는 핵심 요인은 무엇인가?"
- 연구 질문 설정 (RQ1, RQ2, RQ3)
- 분석 범위 결정

**출력물:**
- 문제 정의 문서
- 연구 질문 리스트

**참고 문서**: [NBA_문제정의_및_가설.md](./NBA_문제정의_및_가설.md)

### 2단계: 가설 설정

**목적**: 검증 가능한 가설 수립

**주요 활동:**
- 가설 1: 슛 효율성 가설
- 가설 2: 어시스트 가설
- 가설 3: 승/패 지표 차이 가설
- 가설 4: 시계열 예측 가설
- 가설 5: 클러스터 유효성 가설

**출력물:**
- 가설 리스트
- 검증 방법 계획

**참고 문서**: [NBA_문제정의_및_가설.md](./NBA_문제정의_및_가설.md)

### 3단계: 데이터 수집 및 전처리

**목적**: 분석 가능한 형태로 데이터 준비

**주요 활동:**
1. 데이터 로드
   - 경기 정보, 경기 상세 기록, 선수 정보, 팀 정보
2. 데이터 병합
   - 4개 데이터 소스 통합
3. 데이터 변환
   - MIN 컬럼 변환 (MM:SS → float)
   - 날짜 타입 변환
4. 결측치 처리
   - 숫자형: 0으로 채움
   - 범주형: 'UNKNOWN'으로 채움
5. 이상치 제거
   - MIN < 0 또는 MIN > 60 제거
   - 무기록 행 제거
6. 파생 지표 계산
   - PPM, SEI, EFG%, TS% 등

**출력물:**
- 전처리된 데이터 (PKL, CSV)
- 전처리 통계 리포트

**실행 방법:**
```python
from NBA_전처리_모듈 import NBADataPreprocessor

preprocessor = NBADataPreprocessor()
df = preprocessor.process()
```

### 4단계: 탐색적 데이터 분석 (EDA)

**목적**: 데이터의 특성과 패턴 파악

**주요 활동:**
1. 분포 분석
   - 주요 지표의 분포 확인
   - 정규성 검토
2. 이상치 탐지
   - IQR 방법 또는 Z-score 방법
   - 박스플롯 시각화
3. 상관관계 분석
   - 지표 간 상관관계 히트맵
   - PLUS_MINUS와의 상관관계
4. 다변량 분석
   - 산점도 분석
   - 주요 지표 간 관계 파악

**출력물:**
- 분포 분석 그래프
- 이상치 탐지 결과
- 상관관계 히트맵
- 다변량 분석 그래프

**실행 방법:**
```python
from NBA_EDA_모듈 import NBAEDA

eda = NBAEDA()
eda.perform_all_eda()
```

### 5단계: 통계 검정 (가설 검증)

**목적**: 가설의 통계적 유의성 검증

**주요 활동:**
1. 승/패 그룹 간 차이 검정
   - 독립표본 t-test (정규성 충족 시)
   - Mann-Whitney U 검정 (정규성 미충족 시)
   - 효과 크기 계산 (Cohen's d)
2. 클러스터 간 차이 검정
   - ANOVA (정규성 충족 시)
   - Kruskal-Wallis 검정 (정규성 미충족 시)
   - 사후 검정 (필요 시)
3. 정규성 검정
   - Shapiro-Wilk 검정
   - D'Agostino 검정

**출력물:**
- 통계 검정 결과 테이블
- p-value 및 효과 크기
- 검정 결과 CSV 파일

**실행 방법:**
```python

stat_test = NBAStatisticalTest()
stat_test.perform_all_tests()
```

## 가설 검증 프로세스

### 가설 1: 슛 효율성 가설

**가설**: 슛 효율성(EFG%, TS%)이 단순 득점량보다 승리에 더 큰 영향을 미친다.

**검증 방법:**
1. 특성 중요도 분석 (머신러닝 모델)
2. 상관관계 분석
3. 통계 검정 (회귀 분석)

**결과 해석:**
- 특성 중요도에서 EFG_PCT, TS_PCT가 PTS보다 높으면 가설 지지
- 상관관계에서 EFG%, TS%가 PLUS_MINUS와 더 높은 상관관계를 보이면 가설 지지

---

### 가설 2: 어시스트 가설

**가설**: 어시스트(AST)가 리바운드(REB)보다 승리에 더 큰 영향을 미친다.

**검증 방법:**
1. 특성 중요도 분석
2. 승/패 그룹 간 평균 비교 (t-test)
3. 효과 크기 계산

**결과 해석:**
- 특성 중요도에서 AST가 REB보다 높으면 가설 지지
- 승리 그룹의 AST 평균이 패배 그룹보다 유의하게 높으면 가설 지지

---

### 가설 3: 승/패 지표 차이 가설

**가설**: 승리 경기와 패배 경기 간 핵심 지표에 통계적으로 유의한 차이가 있다.

**검증 방법:**
1. 독립표본 t-test
2. Mann-Whitney U 검정
3. 효과 크기 (Cohen's d)

**결과 해석:**
- p < 0.05이면 귀무가설 기각, 대립가설 채택
- Cohen's d > 0.8이면 큰 효과 크기

---

### 가설 4: 시계열 예측 가설

**가설**: 선수의 최근 5경기 평균 성과가 다음 경기 예측에 유의한 영향을 미친다.

**검증 방법:**
1. 회귀 분석 (특성 중요도)
2. 상관관계 분석
3. 모델 성능 비교

**결과 해석:**
- 최근 5경기 평균 특성이 높은 중요도를 가지면 가설 지지
- 모델 성능이 향상되면 가설 지지

---

### 가설 5: 클러스터 유효성 가설

**가설**: 선수 클러스터 간 성과 지표에 통계적으로 유의한 차이가 있다.

**검증 방법:**
1. ANOVA
2. Kruskal-Wallis 검정
3. 사후 검정

**결과 해석:**
- p < 0.05이면 클러스터 간 유의한 차이 존재
- 사후 검정으로 어떤 클러스터 간 차이가 있는지 확인

## 결과 해석 가이드

### 통계 검정 결과 해석

#### p-value 해석
- **p < 0.001**: 매우 유의한 차이
- **p < 0.01**: 유의한 차이
- **p < 0.05**: 유의한 차이 (일반적 기준)
- **p >= 0.05**: 유의한 차이 없음

#### 효과 크기 해석 (Cohen's d)
- **|d| < 0.2**: 무시할 수 있는 효과
- **0.2 <= |d| < 0.5**: 작은 효과
- **0.5 <= |d| < 0.8**: 중간 효과
- **|d| >= 0.8**: 큰 효과

### 머신러닝 결과 해석

#### 모델 성능 지표
- **분류 (Accuracy)**: 0.7 이상이면 양호
- **회귀 (R²)**: 0.6 이상이면 양호
- **RMSE**: 작을수록 좋음

#### 특성 중요도
- 중요도가 높은 특성이 예측에 더 큰 영향을 미침
- 상위 5개 특성에 집중
