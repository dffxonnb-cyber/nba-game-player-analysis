# 재현성 및 검증 가이드

## 공개 저장소 기준 재현 가능한 범위

| 범위 | 확인 방법 | 비고 |
|------|-----------|------|
| 전처리 | [NBA_전처리_모듈.py](../NBA_전처리_모듈.py) | 경기/선수/팀 병합, 파생 변수 생성 |
| 탐색 분석 | [NBA_EDA_모듈.py](../NBA_EDA_모듈.py) | 분포, 이상치, 상관관계 검토 |
| 통계 검정 | [NBA_통계검정_모듈.py](../NBA_통계검정_모듈.py) | t-test, Mann-Whitney, ANOVA, Kruskal-Wallis |
| 머신러닝 | [NBA_머신러닝_모듈.py](../NBA_머신러닝_모듈.py) | 승패 분류, 선수 회귀, K-Means 군집 |
| 시각화 | [outputs/](../outputs/) 및 [NBA_시각화_모듈.py](../NBA_시각화_모듈.py) | 결과 PNG와 생성 코드 확인 |

## 재현 불가능한 범위

- 저장소만 클론한 직후 동일 결과 확인
- 원본 CSV 없이 전처리와 학습 재실행

즉, 이 저장소는 `비공개 데이터 프로젝트`가 아니라 `공개 Kaggle 데이터 다운로드가 선행되어야 하는 프로젝트`입니다.

## 재현 절차

1. `pip install -r requirements.txt`
2. Kaggle에서 `games.csv`, `games_details.csv`, `players.csv`, `teams.csv` 다운로드
3. `data/` 폴더에 배치
4. `python NBA_통합_파이프라인.py` 실행

## 검증 포인트

- 승리 요인 해석: `eFG%`, `TS%`, `AST`가 핵심 변수로 반복 등장
- 선수 예측: 문서 기준 `R² > 0.6`
- 예측-실측 상관: 약 `0.68~0.72`
- 평균 오차: PLUS_MINUS `±2.5`, PTS `±3.2`
- 군집 결과: K-Means `5그룹`
